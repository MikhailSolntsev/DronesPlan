# C# Code Style

## DateTime

- **Приоритет: `DateTimeOffset`** — использовать везде, где важен момент времени (timestamps, события, логи, синхронизация).
- `DateTime` допустим только для расписаний, не зависящих от часового пояса (например, "каждый день в 09:00" без привязки к TZ).
- `DateOnly` — для календарных дат без времени (дни в календаре, даты рождения, плановые назначения на день).
- `TimeOnly` — для времени суток без даты.

## Общие принципы

- Следовать существующим конвенциям в файле/проекте.
- Явные типы на границах API, `var` внутри методов где тип очевиден.
- Nullable reference types включены, использовать `?` для nullable.
