## Jira Snapshot Planner

Коротко: внутренний веб‑сервис на .NET, который регулярно делает \"тупые\" снапшоты задач из Jira и показывает их как приоритезированный список с текущим ответственным, плюс упрощённый календарный вид.

- **Цель**: убрать ручную работу в Excel по обновлению статусов, исполнителей и сортировке приоритетов, сохранив привычный способ планирования тимлида (приоритеты 0–100 и визуальное распределение по дням).
- **Источник правды**: Jira (on‑premise) — статусы, assignee, базовая информация о задаче.
- **MVP-функции**:
  - Подтягивание задач по заданному JQL/фильтру из Jira.
  - Вычисление приоритета 0–100 на основе статуса и типа задачи.
  - Исключение закрытых задач из активного списка.
  - Отображение списка задач, отсортированных по приоритету по убыванию.
  - Показ текущего исполнителя (assignee) в момент последней синхронизации.
  - Упрощённый календарный вид занятости по задачам/сотрудникам на основе текущего снапшота.

- **Роль тимлида**:
  - Определяет правила маппинга статусов Jira в приоритеты.
  - Решает, какие задачи входят в отбор (JQL/фильтры).
  - Продолжает планировать вручную распределение задач по дням, используя визуализацию системы вместо Excel.

- **Дальнейшее развитие** (за рамками первичного MVP):
  - Учёт отпусков и их наложения на занятость.
  - Дополнительные отчёты по сотрудникам.
  - Автоматические подсказки по вероятным срокам завершения.
